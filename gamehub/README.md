# GameHub - Production-Ready Game Discovery Platform

## Quick Setup

1. Create project at supabase.com
2. Run SUPABASE_SETUP.sql in SQL Editor  
3. Edit js/supabaseClient.js with your URL and anon key
4. Open index.html or run: npx serve .

## Deploy to Netlify
Drag the gamehub folder to netlify.com/drop

## Features: Auth, Ratings, Comments, Bookmarks, Search, Filters, Pagination, Dark/Light Mode, Realtime, RLS Security
