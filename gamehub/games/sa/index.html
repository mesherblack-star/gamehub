🏫 PETUALANGAN KE SEKOLAH

Game Platformer ala Super Mario Bros!
Selamat datang di Petualangan ke Sekolah! Rasakan sensasi bermain ala Mario Bros dengan suara lompat "BOING!" dan suara koin "TUING TUING!". Kumpulkan koin, hindari robot, capai sekolah sebelum waktu habis!

🎮 KONTROL PERMAINAN

🖱️ KEYBOARD

A / D = Bergerak Kiri / Kanan
W / Spasi = Melompat (BOING!)
R + Ctrl = Reset Game

🎮 JOYSTICK / GAMEPAD

Analog Kiri = Bergerak
Tombol A / X = Melompat (BOING!)
Tombol Start = Reset Game

Nama akan ditampilkan selama permainan (maksimal 15 karakter)
Pilih salah satu karakter untuk memulai petualangan
Hubungkan joystick untuk kontrol alternatif
